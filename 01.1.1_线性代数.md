## 一、为什么大模型必须先过线性代数这关

先看一个你一定会反复见到的公式——Scaled Dot-Product Attention：

$$
\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V
$$

其中每个字母背后的含义是：$Q, K, V$ 是三个矩阵，$QK^T$ 是矩阵乘法，$/\sqrt{d_k}$ 是标量除法，softmax 是对每行做归一化，最后再乘以 $V$ 得到输出矩阵。**如果你对这些运算的维度、几何意义、数值稳定性没有直觉，你永远只会"抄公式"而不会"理解公式"。**

再看一个：LoRA 的核心思想是把权重更新 $\Delta W \in \mathbb{R}^{d \times d}$ 分解为 $\Delta W = BA$，其中 $B \in \mathbb{R}^{d \times r}$、$A \in \mathbb{R}^{r \times d}$、$r \ll d$。这里"低秩"的概念直接来自 SVD。

线性代数不是"高数的一个分支"，它是**深度学习的母语**。本节覆盖大模型领域真正会用到的子集，不追求数学系的严谨性，只追求"看论文不卡壳"。

## 二、核心概念

### 2.1 向量（Vector）

一个 $n$ 维向量是 $n$ 个数的有序列表：$\mathbf{x} = [x_1, x_2, \dots, x_n]^T \in \mathbb{R}^n$。

**在大模型中的三个角色**：

1. **Embedding**：每个 token（"猫"、"dog"、"。"）被映射成一个固定维度的向量（如 4096 维）。语义相近的词在这个空间里距离相近。
2. **隐藏状态（Hidden State）**：Transformer 每一层处理完后，每个位置输出一个向量。
3. **参数**：偏置项 $\mathbf{b}$、RMSNorm 的缩放向量 $\mathbf{g}$ 等都是向量。

**向量范数**：衡量向量"长度"。
- L2 范数（欧几里得范数）：$\|\mathbf{x}\|_2 = \sqrt{\sum_i x_i^2}$ — 最常用
- L1 范数：$\|\mathbf{x}\|_1 = \sum_i |x_i|$ — 用于稀疏化（如 L1 正则）
- L∞ 范数：$\|\mathbf{x}\|_\infty = \max_i |x_i|$ — 梯度裁剪常用

**内积（点积）**：$\mathbf{x} \cdot \mathbf{y} = \sum_i x_i y_i = \|\mathbf{x}\|\|\mathbf{y}\|\cos\theta$

这是 Attention 里"相似度"的数学本质。两个向量方向越一致，内积越大；正交则为 0。

**余弦相似度**：$\cos\theta = \frac{\mathbf{x}\cdot \mathbf{y}}{\|\mathbf{x}\|\|\mathbf{y}\|}$，RAG 里检索相似文档就靠它。

### 2.2 矩阵（Matrix）

一个 $m \times n$ 矩阵是 $m$ 行 $n$ 列的数表：$A \in \mathbb{R}^{m\times n}$。

**大模型里的矩阵无处不在**：

- 一层 Linear 层的权重 $W \in \mathbb{R}^{d_{\text{out}} \times d_{\text{in}}}$
- 一个 batch 的 token embedding：$X \in \mathbb{R}^{B \times L \times d}$（这是 3 阶张量，但每个切片都是矩阵）
- 注意力分数矩阵 $S = QK^T \in \mathbb{R}^{L \times L}$

**关键理解：矩阵 = 线性变换**

这是线性代数最重要的几何直觉。一个矩阵 $A$ 作用在向量 $\mathbf{x}$ 上得到 $\mathbf{y} = A\mathbf{x}$，这相当于把 $\mathbf{x}$ 所在空间做了"旋转 + 拉伸 + 剪切"的复合变换。神经网络的每一层 `Linear(x)` 就是在做这件事，再接一个非线性激活函数。

**矩阵乘法**：$(AB)_{ij} = \sum_k A_{ik} B_{kj}$

记住三个等价的理解方式：
1. **逐元素**：$C_{ij}$ 是 $A$ 的第 $i$ 行与 $B$ 的第 $j$ 列的内积
2. **列组合**：$AB$ 的每一列，是 $A$ 的列向量的线性组合
3. **外积累加**：$AB = \sum_k \mathbf{a}_k \mathbf{b}_k^T$（$\mathbf{a}_k$ 是 $A$ 的第 $k$ 列，$\mathbf{b}_k^T$ 是 $B$ 的第 $k$ 行）

第三种理解是 SVD 和 LoRA 的基础，后面会用到。

### 2.3 转置、逆、伪逆

- **转置**：$A^T_{ij} = A_{ji}$。注意力里 $QK^T$ 的 $K^T$ 就是转置。
- **逆矩阵**：只有方阵且满秩的矩阵有逆 $A^{-1}$，满足 $AA^{-1} = I$。**神经网络训练中几乎从不直接求逆**（计算成本 $O(n^3)$、数值不稳定）。
- **伪逆（Moore-Penrose）**：非方阵用 $A^+ = (A^TA)^{-1}A^T$（列满秩时）。最小二乘法的解就是 $\mathbf{x}^* = A^+\mathbf{b}$。

### 2.4 特殊矩阵

| 类型 | 定义 | 大模型中出现的地方 |
|------|------|------------------|
| 对角矩阵 | 只有对角线非零 | SVD 的 $\Sigma$、LayerNorm 的缩放 |
| 单位矩阵 $I$ | 对角全 1 | 残差连接 $y = x + f(x) = (I+f)(x)$ |
| 对称矩阵 | $A = A^T$ | 协方差矩阵、$A^TA$ |
| 正交矩阵 | $Q^TQ = I$ | 旋转矩阵、RoPE 位置编码 |
| 正定矩阵 | 所有特征值 > 0 | Hessian 矩阵、协方差矩阵 |

### 2.5 秩（Rank）—— LoRA 的灵魂

矩阵的秩 = 线性无关的列（或行）的最大数量 = 矩阵的"信息量"。

- **满秩**：$\text{rank}(A) = \min(m, n)$，矩阵没有"冗余"
- **低秩**：$\text{rank}(A) \ll \min(m, n)$，矩阵可以被"压缩表达"

**关键定理**：一个秩为 $r$ 的 $m\times n$ 矩阵可以写成 $A = BC$，其中 $B \in \mathbb{R}^{m\times r}$、$C \in \mathbb{R}^{r\times n}$。原矩阵参数 $mn$，分解后参数 $(m+n)r$。当 $r$ 很小时，压缩非常可观。

**LoRA 的全部数学基础就是这一句话**：假设微调带来的权重变化 $\Delta W$ 是低秩的，那么不用直接训练 $\Delta W$（$d\times d$ 个参数），而是训练 $B\in\mathbb{R}^{d\times r}$ 和 $A\in\mathbb{R}^{r\times d}$（只有 $2dr$ 个参数）。当 $d=4096$、$r=8$ 时，参数量从 1600 万降到 6.5 万，压缩 256 倍。

## 三、特征值分解与 SVD

### 3.1 特征值与特征向量

对方阵 $A$，如果存在非零向量 $\mathbf{v}$ 和标量 $\lambda$ 使得：

$$
A\mathbf{v} = \lambda \mathbf{v}
$$

则称 $\mathbf{v}$ 为特征向量，$\lambda$ 为对应的特征值。

**几何含义**：$\mathbf{v}$ 是经过 $A$ 变换后方向不变（只被拉伸 $\lambda$ 倍）的那些特殊方向。这些"主轴"告诉我们矩阵变换的本质。

**特征分解**：如果 $A \in \mathbb{R}^{n\times n}$ 有 $n$ 个线性无关的特征向量，则 $A = V\Lambda V^{-1}$，其中 $\Lambda$ 是对角矩阵（对角元为特征值），$V$ 的列是特征向量。

**局限**：只对方阵成立，且不一定能分解。这就引出了 SVD。

### 3.2 奇异值分解（SVD）—— 必须掌握

任意 $m\times n$ 实矩阵 $A$（不论形状、秩）都能分解为：

$$
A = U\Sigma V^T
$$

- $U \in \mathbb{R}^{m\times m}$：左奇异向量，正交矩阵，是 $AA^T$ 的特征向量
- $V \in \mathbb{R}^{n\times n}$：右奇异向量，正交矩阵，是 $A^TA$ 的特征向量
- $\Sigma \in \mathbb{R}^{m\times n}$：对角矩阵（通常非方形），对角元 $\sigma_1 \geq \sigma_2 \geq \dots \geq 0$ 是奇异值，等于 $A^TA$ 特征值的平方根

**三大用途**：

**1. 低秩近似（Eckart-Young 定理）**

保留前 $k$ 大奇异值得到 $A_k = U_{:,:k}\Sigma_{:k,:k}V_{:,:k}^T$，这是所有秩 $\leq k$ 矩阵中 Frobenius 范数误差最小的近似。**这就是图像压缩、推荐系统、模型压缩的理论基础。**

**2. LoRA 的理论依据**

LoRA 论文假设预训练后的权重更新 $\Delta W$ 有"低内在秩（intrinsic rank）"。用 SVD 视角看，就是 $\Delta W$ 的前几个奇异值已经捕捉了绝大部分信息。训练 $B$ 和 $A$ 等价于在学习这个低秩子空间。

**3. 主成分分析（PCA）**

对数据矩阵 $X$（行为样本、列为特征），$X$ 的右奇异向量就是主成分方向。

### 3.3 Python 代码：用 SVD 做低秩近似

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成一个"低秩 + 噪声"的数据矩阵
np.random.seed(42)
rank_true = 5
m, n = 100, 80
B_true = np.random.randn(m, rank_true)
C_true = np.random.randn(rank_true, n)
A = B_true @ C_true + 0.1 * np.random.randn(m, n)  # 低秩 + 噪声

# SVD 分解
U, S, Vt = np.linalg.svd(A, full_matrices=False)
print(f"前 10 个奇异值: {S[:10]}")

# 保留前 k 个奇异值的低秩近似
def low_rank_approx(U, S, Vt, k):
    return U[:, :k] @ np.diag(S[:k]) @ Vt[:k, :]

# 查看不同秩下的重建误差
for k in [1, 3, 5, 10, 20, 50]:
    A_approx = low_rank_approx(U, S, Vt, k)
    err = np.linalg.norm(A - A_approx, 'fro') / np.linalg.norm(A, 'fro')
    params_original = m * n
    params_low_rank = k * (m + n)
    print(f"k={k:2d}  相对误差={err:.4f}  参数压缩={params_original/params_low_rank:.1f}x")
```

运行后你会看到，**当 $k=5$（真实秩）时误差已经很小**，继续加大 $k$ 反而在拟合噪声。这就是"低秩先验"的威力。

### 3.4 PyTorch 代码：手写一个最简 LoRA 层

```python
import torch
import torch.nn as nn

class LoRALinear(nn.Module):
    def __init__(self, in_features, out_features, rank=8, alpha=16):
        super().__init__()
        # 冻结的原始权重（预训练得到）
        self.W = nn.Linear(in_features, out_features, bias=False)
        for p in self.W.parameters():
            p.requires_grad = False
        
        # LoRA 的两个低秩矩阵
        self.A = nn.Parameter(torch.randn(rank, in_features) * 0.01)
        self.B = nn.Parameter(torch.zeros(out_features, rank))  # 零初始化保证起点等于原模型
        self.scale = alpha / rank
    
    def forward(self, x):
        # 原始前向 + 低秩增量
        return self.W(x) + (x @ self.A.T @ self.B.T) * self.scale

# 参数量对比
d_in, d_out, r = 4096, 4096, 8
orig_params = d_in * d_out
lora_params = r * (d_in + d_out)
print(f"原始: {orig_params:,}   LoRA: {lora_params:,}   压缩 {orig_params/lora_params:.0f}x")
# 输出：原始: 16,777,216   LoRA: 65,536   压缩 256x
```

## 四、矩阵求导：反向传播的数学语言

深度学习的反向传播本质上是**矩阵函数的链式法则**。

**三个必记的求导公式**（分母布局）：

| 表达式 | 导数 |
|--------|------|
| $\mathbf{y} = A\mathbf{x}$ | $\frac{\partial \mathbf{y}}{\partial \mathbf{x}} = A^T$ |
| $y = \mathbf{x}^T A \mathbf{x}$ | $\frac{\partial y}{\partial \mathbf{x}} = (A + A^T)\mathbf{x}$ |
| $y = \mathbf{a}^T \mathbf{x}$ | $\frac{\partial y}{\partial \mathbf{x}} = \mathbf{a}$ |

**应用：推导 Linear 层的梯度**

前向：$\mathbf{y} = W\mathbf{x} + \mathbf{b}$，损失 $L$。

由链式法则：
- $\frac{\partial L}{\partial \mathbf{x}} = W^T \frac{\partial L}{\partial \mathbf{y}}$
- $\frac{\partial L}{\partial W} = \frac{\partial L}{\partial \mathbf{y}} \mathbf{x}^T$
- $\frac{\partial L}{\partial \mathbf{b}} = \frac{\partial L}{\partial \mathbf{y}}$

这就是 PyTorch 里 `nn.Linear` 的 backward 实现。理解它你就理解了 autograd 的一半。

## 五、常见陷阱与面试高频点

1. **矩阵乘法不满足交换律**：$AB \neq BA$（一般情况）。但满足结合律 $A(BC) = (AB)C$ — **这点对优化 Attention 计算很关键**，例如 $QK^TV$ 按不同顺序计算，内存和算力差别巨大。

2. **浮点数值稳定性**：
   - 不要直接算 $A^{-1}$，用 `np.linalg.solve` 或 LU 分解
   - softmax 要先减去最大值：$\text{softmax}(x_i) = \frac{e^{x_i - \max}}{\sum e^{x_j - \max}}$
   - 注意力除以 $\sqrt{d_k}$ 正是为了防止 $QK^T$ 进入 softmax 饱和区

3. **维度确认**：看论文时第一件事是把每个符号的形状写在旁边。$Q\in \mathbb{R}^{L\times d_k}$、$K\in \mathbb{R}^{L\times d_k}$、$QK^T \in \mathbb{R}^{L\times L}$、最终输出 $\in \mathbb{R}^{L\times d_v}$。

4. **行向量 vs 列向量**：数学习惯列向量（$\mathbf{x}\in\mathbb{R}^{n\times 1}$），但深度学习框架习惯行向量（batch 维在前）。PyTorch 里 `nn.Linear(in, out)` 的权重形状是 `(out, in)`，前向是 `x @ W.T`，不是数学书上的 $W\mathbf{x}$。

5. **秩 vs 维度**：一个 $1000\times 1000$ 的矩阵维度是 1000，但秩可能只有 5。深度神经网络的权重矩阵经验上**有效秩往往远低于维度**，这是低秩方法能 work 的经验基础。
