# LLM 基础：大模型理论与应用实践学习笔记

一套系统化的中文学习笔记，覆盖从**数学与编程基础**，到 **Transformer / 预训练 / 后训练对齐**，再到 **Prompt、RAG、Agent、多模态、推理部署与前沿方向**的完整知识链路。每篇均为独立的 Markdown 文档，便于按需阅读，也可循序渐进通读。

> 完整知识体系与扩展阅读路径见 [AI 大模型学习大纲](AI大模型学习大纲.md)。

## 学习路径建议

基础数学与深度学习 → Transformer 架构 → 预训练 / 后训练对齐 → 应用层（Prompt / RAG / Agent / 多模态）→ 推理优化与部署 → 安全与前沿。每学一章，尽量跑通一个最小可运行示例，理论结合实践。
